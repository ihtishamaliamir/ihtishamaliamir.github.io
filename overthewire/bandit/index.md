## 🛠️ Common Commands Reference

Here are the commands I keep using throughout Bandit:

| Command | Purpose | Example |
|---------|---------|---------|
| `ssh` | Remote connection | `ssh user@host -p 2220` |
| `ls` | List files | `ls -la` |
| `cat` | View file | `cat filename` |
| `find` | Search for files | `find . -name "*.txt"` |
| `grep` | Search text | `grep "password" file.txt` |
| `file` | Identify file type | `file data.bin` |
| `xxd` | Hex dump | `xxd -r data.txt > data.bin` |
| `gunzip` | Decompress | `gunzip file.gz` |
| `tar` | Archive | `tar -xf archive.tar` |
| `nc` | Netcat | `nc localhost 30000` |
| `openssl` | SSL/TLS | `openssl s_client -connect host:port` |
| `nmap` | Port scanning | `nmap -p 31000-32000 localhost` |

### Networking
- SSH connections and authentication
- Port scanning with nmap
- SSL/TLS connections

- File navigation (`ls`, `cd`)

- File manipulation (`mv`, `cp`, `rm`)

- Identifying file types (`file`)
- Hex dumps (`xxd`)

- SSH key authentication
- SSL/TLS certificates
- Port security
- Service identification
